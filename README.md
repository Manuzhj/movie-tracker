# Movie Tracker

A full-stack movie tracking website built from scratch.

## Features
- Sign up and sign in
- Browse films
- Write journal entries for movies you watched
- Rate movies 1-5 stars

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: SQLite
- Security: bcrypt password hashing

## How to run
1. Clone the repo
2. Open the backend folder: `cd backend`
3. Run `npm install`
4. Run `node server.js`
5. Open `frontend/JS.html` in your browser
